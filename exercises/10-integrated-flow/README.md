# 10. Integrated Flow

## :movie_camera: Scenario
Vending machine. Validate selection, handle invalid input loop, check payment, dispense product, and complete flow.

## :memo: Instructions
1. Open `app.js` in this exercise folder.
2. Use this **[Mermaid Editor](https://waficmikati.github.io/mermaid/)** to create your diagram.
3. Keep node ids (`A`, `B`, `C`, ...) and edges aligned with the rubric requirements.
4. Do not rename `answer` or `module.exports` in `app.js`.

## :bulb: Hint
- Your graph should include these key concepts: **start**, **input**, **valid**, **invalid**, **home**, **output**, **end**.
- Aim for at least 7 edges in total before running tests.
- Use branch labels exactly as expected: **yes**, **no**, **valid**, **invalid**.
- Keep names and labels simple to match the validator.
