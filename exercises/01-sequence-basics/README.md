# 1. Sequence Basics

## :movie_camera: Scenario
You prepare coffee before class. Define the following linear flow:

>Start -> Boil water -> Brew coffee -> Serve -> End

There should not be any decisions.

## :memo: Instructions
1. Open `app.js` in this exercise folder.
2. Use this **[Mermaid Editor](https://waficmikati.github.io/mermaid/)** to create your diagram.
3. Keep node ids (`A`, `B`, `C`, ...) and edges aligned with the rubric requirements.
4. Do not rename `answer` or `module.exports` in `app.js`.

## :bulb: Hint
- Your graph should include these key concepts: **start**, **end**.
- Aim for at least 3 edges in total before running tests.
- This exercise does not require branch condition labels.
- Avoid forbidden patterns in the diagram text: while, for.
