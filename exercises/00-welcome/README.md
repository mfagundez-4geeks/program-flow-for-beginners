# Welcome: How This LearnPack Works

In this project, each exercise asks you to solve a logic scenario by submitting a Mermaid flowchart.

## :computer: Where you write your solution

Edit only `app.js` inside each exercise folder.

## :hammer: Workflow

1. Use this tool created by a 4Geeks graduate student: **[Mermaid Editor](https://waficmikati.github.io/mermaid/)**
2. Build your diagram there.
3. Copy and paste the Mermaid code into `app.js`.

## :bulb: Tips
- You do not need to master Mermaid syntax in this tutorial.
- Focus on the logic requested by each exercise.
- If tests fail, compare your result with `rubric.json`.
- You can refer to the Mermaid Flowchart syntax here: **[Official Mermaid Docs](https://mermaid.ai/open-source/syntax/flowchart.html)**
